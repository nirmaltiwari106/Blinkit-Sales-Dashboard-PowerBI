# 🛒 BlinkIT Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard built using BlinkIT grocery data. The dashboard provides insights into sales performance, outlet distribution, product categories, and customer behavior.

---

## 📊 Key Features
- Total Sales Analysis
- Average Sales & Ratings
- Number of Items Sold
- Outlet Establishment Trends
- Outlet Size & Location Insights
- Item Type Distribution
- Fat Content Analysis

---

## 📈 Dashboard Insights
- 💰 Total Sales: 1M+
- ⭐ Average Rating: 3.9
- 📦 Total Items: 8,523
- 🏪 Tier 3 outlets contribute highest sales
